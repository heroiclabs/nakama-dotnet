\hypertarget{md_codegen__r_e_a_d_m_e_autotoc_md82}{}\doxysubsection{Rationale}\label{md_codegen__r_e_a_d_m_e_autotoc_md82}
We want to maintain a simple lean low level client within our C\# client which has minimal dependencies so we built our own. This gives us complete control over the dependencies required and structure of the code generated.

The generated code is designed to be supported within Unity engine, Xamarin, Godot engine, and other projects. It requires .NET4.\+5 framework, Tiny\+Json, and uses {\ttfamily System.\+Threading.\+Tasks}.\hypertarget{md_codegen__r_e_a_d_m_e_autotoc_md83}{}\doxysubsection{Limitations}\label{md_codegen__r_e_a_d_m_e_autotoc_md83}
The code generator has {\bfseries{only}} been checked against the Swagger specification generated for \mbox{\hyperlink{namespace_nakama}{Nakama}} server. YMMV.
