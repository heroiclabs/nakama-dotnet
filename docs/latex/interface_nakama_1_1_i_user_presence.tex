\doxysubsection{Detailed Description}
An object which represents a connected user in the server.  

The server allows the same user to be connected with multiple sessions. To uniquely identify them a tuple of {\ttfamily \{ node\+\_\+id, user\+\_\+id, session\+\_\+id \}} is used which is exposed as this object.
